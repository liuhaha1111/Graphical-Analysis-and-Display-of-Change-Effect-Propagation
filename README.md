# 变更效应传播图形化分析展示

这是一个前端原型，用于演示产品结构建模、产品设计知识图谱和变更效应传播可视化如何共享同一套数据模型。

## 文档

- [功能说明](./docs/feature-overview.md)
- [操作手册](./docs/user-manual.md)

## 原型范围

- 产品结构建模
  基于共享工作区展示 BOM 层级、组件参数和参数依赖关系。
- 产品设计知识图谱
  将产品组件、供应伙伴和物流路径组织成统一的产品-供应链网络视图。
- 变更效应传播可视化
  支持注入变更场景，并用确定性规则计算传播路径、综合评分、成本风险和工期风险。

## 共享模型

三个功能页共享一份内存态 `WorkspaceModel`：

- 产品域：组件、参数、参数依赖
- 供应链域：供应商、供货关系、物流路径
- 分析域：变更场景、传播结果

当前版本使用内置示例数据驱动演示，不包含 Excel/CSV 导入、后端持久化或多人协作。

## 运行方式

前置条件：

- Node.js

安装依赖：

```bash
npm install
```

启动开发环境：

```bash
npm run dev
```
